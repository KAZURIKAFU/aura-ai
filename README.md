## 🚀 What is AuraAI?

AuraAI is an intelligent personal assistant built with a custom NLP engine, LangChain-style conversation memory, and a suite of built-in tools — all wrapped in a beautiful dark-theme chat interface.

---

## ✨ Features

| Feature | Description |
|---|---|
| 💬 Natural Language Q&A | Answers questions on AI, ML, IoT, Cloud, Cybersecurity |
| 🔢 Math Calculator | Evaluates complex math expressions |
| 🐍 Code Generator | Generates Python code for common tasks |
| 📧 Email Drafting | Creates professional email templates |
| 📝 Text Summarizer | Extracts key points from any text |
| 🧠 Conversation Memory | Remembers last 20 turns of conversation |
| ⚡ Quick Actions | 10 pre-built one-click queries |
| 📊 Session Stats | Live turn count & memory usage tracker |
| 🎨 Dark Theme UI | Professional chat interface |


## ⚙️ Setup

```bash
git clone https://github.com/KAZURIKAFU/aura-ai.git
cd aura-ai
pip install -r requirements.txt
python app.py
# Open: http://localhost:8053
```
